# Exercices de PHP - Partie 3 | Programmation Orientée Objet (POO)

Les exercices proposés visent à approfondir les concepts de programmation orientée objet en PHP. Les exercices commencent par la création d'une classe User avec des propriétés et une méthode, puis progressent vers la création de constructeurs et la manipulation d'objets. Le but de ces exercices est de permettre aux apprenants de mieux comprendre comment créer des objets et interagir avec eux en utilisant PHP. L'utilisation de PDO pour récupérer des données à partir d'une base de données et l'affichage de ces données est également abordée dans l'exercice 5. Les exercices sont conçus pour que les apprenants puissent les compléter progressivement, en ajoutant des fonctionnalités plus avancées à mesure qu'ils avancent.
